## ESCPOS PROJECT
ESC/POS Printer driver for Node.js

## Installation

### Dependency
```bash
"node": v14.15.4
"npm": v6.14.10
"escpos": ^3.0.0-alpha.6,
"escpos-usb": ^3.0.0-alpha.4
```
### git
```bash
$ git clone https://github.com/SorawitSiri/posprinter-js
```

### Linux/Ubuntu
if escpos-usb 
```bash
apt-get install build-essential libudev-dev
```
```bash
$ npm install
```
